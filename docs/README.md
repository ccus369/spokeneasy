# SpokenEasy — 项目启动包

这是 SpokenEasy Android 项目的完整需求/设计文档包。

请依次阅读以下文件，然后开始实现：

1. `CLAUDE.md` — 工作规范与约定（含项目状态、版本纠正）
2. `01-requirements.md` — 功能需求
4. `02-technical-spec.md` — 技术选型与规范
5. `03-architecture.md` — MVVM 架构设计
6. `04-data-schema.md` — 数据库表结构
