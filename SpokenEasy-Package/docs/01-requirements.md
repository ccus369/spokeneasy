# SpokenEasy - 英语口语练习APP 需求文档

## 项目概述
开发一款安卓英语口语练习APP，三大核心板块：单词联想句子、口语连读练习、分级听力训练。

## 技术栈
| 项目 | 选择 |
|------|------|
| 语言 | Kotlin |
| UI 框架 | Jetpack Compose |
| 数据库 | Room (SQLite) + 预置 .db 文件 |
| 导航 | 侧边抽屉 + 底部导航栏 |
| 音频播放 | TTS(第一版) → 后期替换预录音频 |
| 录音 | MediaRecorder |
| AI 评分 | 讯飞 Android SDK (第一版用 Mock 占位) |
| 最低 SDK | 27 (Android 9.0) |
| 目标 SDK | 36 (Android 16) |
| 包名 | com.spokeneasy.app |
| 应用名 | SpokenEasy |

## 功能需求

### 板块1：单词联想句子
- 500个高频单词，每个包含：
  - 单词本身
  - 音标
  - 3个常用口语句子（英文+中文翻译）
- 功能点：
  - 单词列表展示
  - 点击单词显示关联句子
  - 支持隐藏/显示中文翻译
  - 点击句子播放发音（TTS）
  - 跟读功能（录音+AI评分）
  - **完成标准：跟读评分 ≥ 60分**

### 板块2：口语连读练习
- 50个连读例子，每条例包含：
  - 连读规则名称
  - 原始写法（如 "give me"）
  - 连读写法（如 "gimme"）
  - 例句
- 连读类别包括但不限于：
  - wanna/gonna
  - outta (out of)
  - gimme (give me)
  - lemme (let me)
  - dunno (don't know)
  - kinda (kind of)
  - I'm in a / we're in 类
- 功能点：
  - 规则分类展示
  - 点击播放标准发音
  - 跟读练习+AI评分
  - **完成标准：练习评分 ≥ 60分**

### 板块3：分级听力训练
- 3个难度等级（初级/中级/高级）
- 共50段对话
- 每段对话包含：
  - 标题
  - 完整对话文本
  - 音频文件（第一版TTS占位）
  - 3道选择题
- 功能点：
  - 按难度选择对话
  - 播放音频（TTS占位，后期替换）
  - 显示/隐藏文本
  - 答题并得分
  - 记录完成状态
  - **完成标准：答题 ≥ 60分**

## 用户系统（第一版）
- 仅游客模式，设备UUID标识
- 记录：学过的单词、练习过的连读、完成的听力对话和得分、最后学习时间

## 数据来源
- 基础数据由开发者人工整理（500词 + 50连读 + 50对话）
- 预置 .db 文件通过 SQLite 可视化工具创建，放入 assets/database/
- 预留调用大模型API接口（后续扩展）

## 录音+AI评分流程
1. 用户点击跟读按钮
2. 录制用户声音（3-10秒）
3. 调用评分接口（抽象层）
4. 返回评分（0-100）和反馈
5. APP展示评分结果

## 导航结构
- **侧边抽屉**：三个模块入口
- **底部导航栏**：3项 — 主页(学习) / 阅读 / 设置(右侧)
- **设置页**：关于版本、切换语言、学习时长
