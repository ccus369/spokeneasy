# SpokenEasy — 项目启动包

这是 SpokenEasy Android 项目的完整需求/设计文档包。

## 给 Claude 的启动指令

请依次阅读以下文件，然后开始实现：

1. `CLAUDE.md` — 工作规范与约定
2. `SPOKENEASY_STATUS.md` — 项目状态与开发计划（含版本纠正）
3. `docs/01-requirements.md` — 功能需求
4. `docs/02-technical-spec.md` — 技术选型与规范
5. `docs/03-architecture.md` — MVVM 架构设计
6. `docs/04-data-schema.md` — 数据库表结构

## 文件清单

```
SpokenEasy-Package/
├── README.md                   ← 本文件
├── CLAUDE.md                   ← 工作规范
├── SPOKENEASY_STATUS.md        ← 项目状态 + 开发计划 + 版本纠正
└── docs/
    ├── 01-requirements.md      ← 需求文档
    ├── 02-technical-spec.md    ← 技术规范（⚠️ 版本有误，见 STATUS 中的纠正）
    ├── 03-architecture.md      ← 架构设计
    └── 04-data-schema.md       ← 数据库设计
```
